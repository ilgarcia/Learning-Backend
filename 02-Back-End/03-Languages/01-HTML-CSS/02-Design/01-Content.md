### Applying fonts

**Embed**

1. Find a Font
2. Get the embed link
3. Paste the embed link (import) at your stylesheet

```
@import url('font url')
```

**Download**

1. Find a Find
2. Download the font
3. Import at your stylesheet

```
#font-face{
    font-family: ;
    src: url('font path 1') format(), url('font path 2') format();
    font-weight: normal;
    font-style: normal;
}
```

Tipos de format():
- opentype (otf)
- truetype (ttf)
- embedded-opentype
- truetype-aat (Apple Advanced Typography)
- svg

:rotating_light: Prefer embed font because of the cache memory
